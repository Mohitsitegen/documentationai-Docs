## The 30-second pitch

SiteZen converts your Figma sections into fully-functional WordPress blocks. No copy-pasting CSS, no rebuilding layouts, no developer required.

**Design in Figma → Click "Convert" → Live on WordPress.** That's it.

<Callout kind="info" collapsed="false">
  SiteZen is built for agencies, freelancers, and designers who want to ship WordPress sites at design-tool speed.
</Callout>

## What you can do with SiteZen

- **Convert any Figma section** into a WordPress block in 30-60 seconds

- **Pixel-perfect output** — exact colors, fonts, spacing, and images from your Figma file

- **Smart component detection** — tabs, accordions, sliders, pricing tables, forms, and more are automatically recognized and made interactive

- **Mobile responsive by default** — every conversion includes tablet and mobile breakpoints

- **Editable in WordPress** — every element stays editable from the Gutenberg block editor

- **Reusable templates** — save headers, footers, tabs, and accordion bodies as reusable templates

## How it works (the big picture)

<Steps>
  <Step title="Design in Figma" icon="figma" title-type="p">
    You design your page in Figma like you normally would. No special setup required — any Figma file works.
  </Step>

  <Step title="Copy the section link" icon="link" title-type="p">
    Right-click any section frame in Figma → "Copy link to selection" → paste it into SiteZen.
  </Step>

  <Step title="Upload a screenshot" icon="image" title-type="p">
    Drop a screenshot of the section so SiteZen knows exactly what the final layout should look like.
  </Step>

  <Step title="Convert and push" icon="zap" title-type="p">
    Click "Convert this section". In \~30-60 seconds, your design lands as a real, editable WordPress page.
  </Step>
</Steps>

## What you need

- A **WordPress site** (any host — InstaWP, WP Engine, Bluehost, self-hosted, all work)

- A **Figma account** with the file you want to convert

- A **SiteZen account** (free trial available, then paid plans starting at $39/mo)

## Two pieces, one product

SiteZen has two parts that work together:

<Columns cols="2">
  <Card title="The Plugin" icon="puzzle" horizontal="false">
    Installs on your WordPress site. Registers the SiteZen Section block, handles per-element editing, and powers all interactive components.
  </Card>

  <Card title="The Platform" icon="cloud" horizontal="false">
    A web app where you paste Figma URLs, upload screenshots, run conversions, and push results to your WordPress site.
  </Card>
</Columns>

You'll set up both — it takes about 5 minutes total. The [Quick Start guide](/quickstart) walks you through it.
